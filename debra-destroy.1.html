NAME

debra-destroy -- destroy a Debra directory

SYNOPSIS

debra destroy name [-h]

DESCRIPTION

debra-destroy destroys a Debra directory. It is equivalent to the command

rm -rf <name>

and is provided for convenience.

OPTIONS

name
Name of a Debra directory.
-h, --help
Show a help message.

debra-destroy(1) -- destroy a Debra directory
=============================================

## SYNOPSIS

`debra destroy` _name_ [`-h`]  

## DESCRIPTION

`debra-destroy` destroys a Debra directory.  It is equivalent to the command

	rm -rf <name>

and is provided for convenience.

## OPTIONS

* _name_:
  Name of a Debra directory.
* `-h`, `--help`:
  Show a help message.
